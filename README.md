## 🚀 Getting Started

### Prerequisites
To run the programs in this repository, ensure you have the required compilers and environments installed on your machine:
* **GCC Compiler** (for compiling C programs)
* **Python 3.x** (for running Python scripts)
* **MySQL Server** (for executing database queries)

### Execution Guide

#### Running C Programs
Navigate to the target directory and compile the source code using `gcc`:
```bash
gcc filename.c -o output
./output

```
#### Running Python Scripts
Execute the python files directly from the terminal:
```bash
python filename.py

```
#### Running SQL Scripts
Log into your MySQL monitor or run the script directly against your database:
```bash
mysql -u root -p database_name < script.sql

```
